# Memotron v0.61.2
<div align="right">*Aug 23, 2025*</div>
> Rolling now

This update adds support for social media clips which includes the following.
- Bluesky
- Linkedin
- Reddit
- Mastodon
- Threads
- Instagram

:::note
To start clipping on above-mentioned platforms, please use Memotron clipper `v0.58.0` and above.
:::
