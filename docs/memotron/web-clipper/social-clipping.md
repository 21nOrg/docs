# Social clipping

Use our handy `Save to Memotron` button on below listed social media platforms once you install our Clipper extension to quickly clip any social post.

__Supported platforms:__
- X (formerly Twitter)
- Bluesky
- Linkedin
- Reddit
- Mastodon
- Threads
- Instagram
